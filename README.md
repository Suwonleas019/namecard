어려웠던 점 :
  1. 나의 명함 사진 변경을 클릭했을 때 다음 사진으로 바뀌어야 했는데. 버튼이 아닌 사진을 클릭했을 때 사진이 바뀌게 만들었었다.
     나중에 사진 변경 버튼을 따로 만들고 사진을 클릭했을 때 사진이 변경되는 코드를 삭제했음에도 불구하고 사진이 자꾸 사진이 클릭
     했을 때만 변경이 되는 어려움이 있었다.  
  2. 또한 사진 변경을 클릭했을 때 바로 사진이 바뀌었어야 하지만 처음 클릭할 때는 바뀌지 않아서 곤황을 겪었었다.
